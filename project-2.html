Project Overview

Vinted is one of the biggest website and smartphone application to buy or sell clothes and accesories. This was a very good project to develop because it requires multiple functionalities, such as : creation and authentification of accounts using cookies, publication of selling announces, the use of Stripe's API...

The next features that will be implemented are :
- Improve Stripe design feature
- Search-bar function
- Delete publication function
- Better design
- Responsiveness
